
core
	deze map bevat functionaliteiten die aangeroepen worden in index en in de templates, zoals sessions en menu bijvoorbeeld
	Sessions.php zal de logica bevatten om de sessie te starten
	Menu.php de logica om het menu op te bouwen
	Register.php etc. 
	De templates van deze core functionaliteit zit in core/templates
core/functions
	deze map bevat *.functions.php bestanden die allemaal in index.php geladen worden en overal gebruikt kunnen worden. functies worden in categorieen ingedeeld zoals bijv. global.functions.php, output.functions.php
core/templates
	bevat de templates van Register, Login, Admin en Menu en eventuele andere absolute core modules
library (/js-assets, /css-assets, /static)
	deze map bevat alles aan css, js en images dat niet bij een thema behoort en door de core gebruikt wordt.
modules
	bevat alles aan modules, zoals forum, blog, mail, cms etc. etc.
modules/*/
	de hoofdmap van een module
module/forum/functions
	bevat alle specifieke functies voor een module, ook in bestanden zoals topics.forum.php opgedeeld
modules/forum/modifiers
	bevat bestanden zoals Menu.php die invloed uitoefent op Menu.php in /core, menuitems toevoegt etc.
modules/forum/templates
	bevat bestanden zoals boardindex.forum.php die de basis zijn voor de output van onderdelen van de module
themes
	bevat de themamappen
themes/hannover
	bevat o.a. base_template.php, waarop de uiteindelijke output wordt gebasseerd, deze wordt via Output.php in /core en de templates van de modules gevuld
themes/hannover/...
	bevat verder css, images, fonts en js die specifiek zijn voor het thema
cache
	bevat cache bestanden
uploads
	bevat avatars en files
